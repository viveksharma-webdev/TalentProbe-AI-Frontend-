
# 🚀 TalentProbe AI

🔗 Live Demo: https://talent-probe-ai.vercel.app/

TalentProbe AI is a **MERN stack web application powered by Generative AI** that helps users prepare for interviews by generating a **personalized interview strategy** based on their profile and target job.

---

## ✨ Features

- 🔐 User Authentication (Register / Login)
- 📄 Upload Resume (PDF / DOCX)
- 🧠 AI-Powered Analysis using Gen-AI
- 🎯 Job Description Matching
- 📊 Match Score Calculation
- ⚠️ Skill Gap Identification
- 💻 Technical Interview Questions
- 🗣️ Behavioral Interview Questions
- 🗺️ Personalized Preparation Roadmap
  - Day-wise plan
  - Focus areas
  - Tasks for each day
- 📥 Download Resume (Processed)

---

## 🧠 How It Works

1. User logs in / registers
2. Provides:
   - Job Description
   - Resume (optional)
   - Self Description (optional)
3. AI analyzes:
   - Skills
   - Experience
   - Job requirements
4. Generates:
   - Match Score (%)
   - Skill Gaps
   - Technical Questions
   - Behavioral Questions
   - Preparation Plan (Day-wise roadmap)
